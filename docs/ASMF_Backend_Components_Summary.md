# Сводная таблица компонентов ASMF-AI-Agent Backend

| Папка | Файл | Назначение | Класс/Модуль | Ключевые возможности |
|-------|------|------------|--------------|---------------------|
| **ai-core** | conversation.js | Управление диалогами | ConversationManager | • Многосессионные диалоги<br>• Контекстное отслеживание<br>• Анализ настроений<br>• Статистика разговоров<br>• Интеграция с памятью |
| **ai-core** | nlp-processor.js | Обработка естественного языка | NLPProcessor | • Поддержка 3 языков (EN/RU/UA)<br>• Извлечение сущностей<br>• Анализ намерений<br>• Семантический анализ<br>• Параллельная обработка |
| **ai-core** | training.js | Обучение и анализ документов | TrainingProcessor | • Поддержка 7 форматов файлов<br>• Извлечение знаний<br>• Пакетная обработка<br>• Экспорт в JSON/CSV/XML |
| **asmf-engine** | memory-core.js | Ядро памяти ASMF | ASMFEngine | • 3-слойная архитектура памяти<br>• Автономное обучение<br>• Семантическое сходство<br>• Временная консолидация<br>• Сжатие памяти |
| **google-drive** | storage.js | Интеграция с Google Drive | DriveStorage | • API аутентификация<br>• Управление папками<br>• Резюмируемая загрузка<br>• Резервное копирование<br>• Лимиты до 15GB |
| **config** | credentials.example.json | Шаблон конфигурации | JSON Schema | • Инструкции настройки<br>• Структура учетных данных<br>• Рекомендации безопасности<br>• Лимиты хранилища |
